# PART 2 — SCALE MODEL (build this first)

> Goal: an almost-exact replica you can hold in your hands — to **prove geometry, the heater
> silhouette, glow placement and packaging** before cutting steel. Pick a **scale** and a **build
> track**. You can run more than one track to experiment.

**Scale:** start **1:5** (≈ 470 mm long for a 2350 mm cart) — big enough to be detailed and carryable.
1:6–1:8 if you want it pocketable; 1:4 if you want to test real-ish joints.

### Track A — Solderable metal (the "weld the model" track) — *most realistic*

| Item | Use | Options |
|------|-----|---------|
| Brass rod 2–3 mm + brass tube (K&S sizes) | Frame members — **silver-solder/braze the joints** (true analog to welding the real frame) | Steel wire/rod + resistance/mini-spot welder |
| Silver solder / brazing rod + flux | The "welds" — strong, fillable joints | Soft solder (weaker, easier) |
| Mini butane torch or resistance solderer | Heat for brazing | Soldering iron (soft solder only) |
| Brass sheet offcuts | Gussets, brackets | Tin/steel shim |
| Files + flux brush + pickle/clean | Joint prep & cleanup | — |

*Why this track:* lets you literally practise welded-joint geometry and test stiffness at model scale — directly de-risks the full-build welding decision.

### Track B — 3D printed — *fastest geometry iteration*

| Item | Use | Options |
|------|-----|---------|
| PLA/PETG filament (or print service) | Frame, shell, wheels, fixtures | Resin print for fine detail |
| CAD model of the cart | Source for prints; reuse for FEA later | — |
| Press-fit pins / small screws | Joining printed parts | CA glue / epoxy |
| Add-in weights | Tune CoG to test anti-tip (C1) | Coins, fishing weights |

*Why:* change wheelbase/track/ride-height in CAD and reprint in an evening — cheapest way to iterate the **geometry decision (T009)**.

### Track C — Foam / styrene / card — *cheapest, fastest for the LOOK*

| Item | Use | Options |
|------|-----|---------|
| Foamboard / XPS foam | Heater body block — test silhouette & crowd read (SP1) | Styrene/plasticard sheet |
| Styrene sheet + plastic cement | Panels, grille bars | Card + PVA |
| Balsa/basswood strip | Quick frame mock-up | Bamboo skewers |
| Spray primer + matte dark + orange | Finish to test the heater + bottle read | Acrylic brush paint |

*Why:* prove the gag and the recognisable heater look for almost nothing before investing in detail.

### Track D — Wood (balsa/basswood) — *classic, glue-jointed*

| Item | Use | Options |
|------|-----|---------|
| Balsa/basswood strip + sheet | Frame and ply-deck analog | Lite-ply |
| CA glue + epoxy | Joints | Wood glue (slower) |
| Pins/clamps | Assembly jig | — |

### Shared model parts (any track)

| Item | Use | Options |
|------|-----|---------|
| Model wheels (RC/diecast) ~40–60 mm | Rolling chassis | 3D-printed wheels · bottle caps (mock-up) |
| Music wire / steel rod 2–3 mm | Axles | Brass rod |
| Micro ball bearings | Free-rolling axles | Plastic tube bushes |
| 3 mm red LEDs + coin cell / AAA holder + switch | **Working mini glow** — test bar-by-bar ignition (SP5) | Addressable micro-LED for sequencing |
| Small magnets | Removable bottle-driver (snap-in like the real gag) | Velcro dots |
| CA glue, epoxy, plastic cement, primer, paint | Assembly & finish | — |
| Modelling tools | Razor saw, hobby knife, mini files, pin vice, helping-hands | — |

## How the two phases connect

1. **Track C** (foam) settles the *look* and the gag → feeds the body-shell + costume choices.
2. **Track B** (3D print) settles the *geometry* → validates the T009 layout numbers and CoG.
3. **Track A** (brass/solder) rehearses the *welded structure* → de-risks the Option-C build and your weld practice.
4. What survives on the model becomes the confirmed spec for **Part 1**, priced into the BOM and budget.
